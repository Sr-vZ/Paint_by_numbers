# Paint_by_numbers

Dependencies:

The Program needs following Programs to work.
1) Python 3 (Download from here: https://www.python.org/downloads/)
2) Chrome/Firefox 
3) Terminal/ Command prompt/ Power Shell/ Shell

Setup:

1) Once Python is installed unzip the folder.
2) Open Terminal/ Command prompt/ Power Shell/ Shell inside the folder or navigate to this folder
3) Run the following commnand : "pip3 install -r requirements.txt" (without the quotes)
4) Once all the dependencies are insatalled run the following command : "python flask_app.py"
5) Open your preferred browser (Chrome/Firefox) and go to : http://127.0.0.1:5000/

How to use the application:
1) Upload the image you want to generate Paint by Numbers for
2) Select the number of colors you want in the resulting image
3) Select detail level of the image higher detail = larger color area
4) Upload the file and wait for the resulting images
